Divi Cookie Consent is a free WordPress plugin for alerting users about the use of cookies on your website.

## Description

Divi Cookie Consent display a cookie notice on your wordpress website and inform the users that your site uses cookies. It's one of the simplest and lightweight plugin in available for wordpress plugin directory. 

[Check the Demo](http://divipeople.com/)
