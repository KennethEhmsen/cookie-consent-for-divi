Divi Cookie Consent is a free WordPress plugin for alerting users about the use of cookies on your website.

== Description ==

Divi Cookie Consent display a cookie notice on your wordpress website and inform the users that your site uses cookies. It's one of the simplest and lightweight plugin in available for wordpress plugin directory. 

[Check the Demo](http://divipeople.com/)

## Features:
* Fully customisable to look just like your own website’s style
* 3 Layouts
* Customizable Cookie Message
* Changle Cookie Popup
* Link to Privacy Policy page

Get in touch for your upcoming project: [Hire Us](https://divipeople.com/services)

== Installation ==
1. Install Cookie Consent for Divi either via the WordPress.org plugin directory or by uploading the files to your server
2. Activate the plugin through the ‘Plugins’ menu in WordPress
3. Go to the Cookie Consent for Divi customizer settings and set your options.
